# landing-gsap

Micro sitio conceptual para el microestudio de animación **NovaFrame Studio**. Presenta servicios y proyectos interactivos con transiciones suaves construidas con [GSAP](https://greensock.com/gsap/).

## Contenido
- `index.html`: estructura principal y secciones del sitio.
- `styles.css`: estilos y layout responsivo.
- `script.js`: animaciones y microinteracciones con GSAP (ScrollTrigger, ScrollToPlugin).

## Cómo ejecutarlo
1. Instala una extensión de servidor estático o utiliza `npx serve` / `python -m http.server` desde la raíz del proyecto.
2. Abre `http://localhost:3000` (o el puerto indicado por tu servidor) en tu navegador.

El sitio usa CDNs de GSAP por lo que no se requiere compilación adicional.
